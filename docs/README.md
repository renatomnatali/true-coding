# Documentacao do Projeto

## Estrutura de Pastas

```
docs/
├── specification/       # Especificacoes do produto (movido de /Spec)
│   ├── mvp.md          # Definicao do MVP
│   └── ...
├── layout/             # Referencias visuais da landing page
├── KNOWLEDGE-BASE.md   # Problemas comuns e solucoes
├── UX-PRINCIPLES.md    # Principios de UX do projeto
├── UX-PROPOSAL-PROJECT-PAGE.md  # Proposta da pagina de projeto
└── trunk-based-development.md   # Guia de desenvolvimento
```

## Prototipos

Prototipos HTML estaticos ficam em `/prototypes/`. Abra diretamente no navegador para visualizar.

```
prototypes/
├── project-page-A.html  # Layout 3 colunas
├── project-page-B.html  # Layout 2 colunas
└── project-page-C.html  # Layout chat centralizado (mobile)
```
