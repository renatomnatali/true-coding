Dicas de Implementacao em React/Tailwind

Componente: WireframePreview
Entrada: wireframes: Array<{"{"} name, description, layout {"}"}>

Logica de deteccao do layout: Fazer layout.toLowerCase() e testar keywords:

  • sidebar -> Grid 2 colunas com barra lateral estreita + area principal
  • card ou grid -> Grid de retangulos (2x2 ou 3 colunas)
  • list ou table -> Linhas empilhadas horizontais
  • form -> Retangulos verticais (campos) + botao
  • map ou mapa -> Area grande + sidebar/panel
  • mobile ou app -> Viewport de telefone (140px wide, border-radius 16px)
  • Default -> Header + body generico

Persona badges: Se o campo wireframes nao tem persona, inferir pela keyword no name ou omitir o badge. Para adicionar persona, enriquecer o JSON adicionando campo persona?: string no prompt.
