Dicas de Implementacao em React/Tailwind

Componente: NavigationPreview
Entrada: navigation: Array<{"{"} name: string, description: string {"}"}>
Logica de deteccao: Fazer name.toLowerCase() e testar keywords:

  • sidebar -> Wireframe de sidebar com menu lateral
  • bottom ou tab bar -> Wireframe de bottom tabs (formato mobile)
  • tabs ou top -> Wireframe de tabs horizontais
  • breadcrumb -> Wireframe de barra de breadcrumbs
  • hamburger ou drawer -> Wireframe de menu hamburger
  • Default -> Texto descritivo com icone generico

Tailwind classes chave: bg-gray-100, bg-gray-200, bg-gray-300 para zonas de wireframe. bg-blue-600 para elementos ativos/highlight. rounded-md ou rounded-lg para bordas. Grid com grid grid-cols-2 gap-4 em desktop, grid-cols-1 em mobile.
