Autorizar Netlify

Vou configurar o deploy automatico no Netlify. Toda vez que eu fizer um commit no GitHub, o Netlify compilara e publicara seu app automaticamente.

O que sera configurado:
  • Site vinculado ao repositorio GitHub
  • Build automatico a cada commit
  • SSL/HTTPS gratuito (Let's Encrypt)
  • CDN global para performance
  • Preview de cada branch/PR
Voce sera redirecionado para o Netlify para autorizar o acesso

Dica: O Netlify oferece 100 GB de banda e 300 minutos de build por mes no plano gratuito. Suficiente para a maioria dos projetos.
